box-shadow（盒子阴影）

内阴影

box-shadow属性值前加inset
box-shadow：inset 10px 20px 30px rgba(0,0,0,.4);

多阴影

box-shadow属性值可以用逗号隔开多个
box-shadow：10px 20px 30px red，10px 20px 30px green，10px 20px 30px blue;
